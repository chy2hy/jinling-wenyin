### 1.3 创作背景

南京云锦，因其绚丽多姿、灿若云霞而得名，至今已有近1600年历史。2009年，南京云锦被联合国教科文组织列入《人类非物质文化遗产代表作名录》。然而，随着现代纺织工业的冲击，云锦的传承面临严峻挑战——传统手工艺人老龄化，年轻人对云锦的认知度低，云锦文化传播方式局限于实物展览和平面图片，缺乏数字化、交互式的传播手段。

本项目以"科技赋能非遗传承"为核心理念，利用Web 3D技术构建一个沉浸式的南京云锦数字化交互展示平台，让用户在浏览器中即可自由探索云锦的纹样之美、织造之精、文化之深。

**作品名称**：南京晓庄学院-佳豪队-金陵纹影：南京云锦数字化3D交互展示

**技术特色**：零安装、零依赖、浏览器即开即用，基于 Three.js r160 纯前端架构，所有纹理由 Canvas 2D 程序化生成，无需任何外部图片资源。

## 六、技术分析

### 6.1 性能优化
- 使用 BufferGeometry 减少 CPU 开销
- 粒子系统使用单次 drawcall 渲染
- 阴影贴图限制在 1024×1024
- 像素比限制 max 2x，平衡画质与性能

### 6.2 视觉设计理念
- **色彩体系**：以金色 (#d4a853) 为主色调，朱红、墨色、古铜为辅，雍容华贵
- **空间布局**：传统围廊式格局，中轴对称，庄重典雅
- **光影氛围**：仿古烛光暖色照明 + Bloom 泛光，再现"灿若云霞"
- **细节装饰**：檐角飞翘、灯笼摇曳、金粉飘洒，身临其境
