## ✅ 提交前必读

### 1️⃣ 作品命名

> **3DShow 标题一经确定不可随意修改**

建议命名格式：
```
南京晓庄学院-佳豪队-金陵纹影：南京云锦数字化3D交互展示
```

### 2️⃣ 组队要求

> **系统自动核验人数，不足3人会拦截提交**

当前团队：**陈鸿宇（队长）** → 还需 **至少2名队员**

建议尽快拉人，对方不需要会技术——这个作品的3D展示部分我全做完了。

队员注册后你在大赛官网 [3DDS.3DDL.net](https://3DDS.3DDL.net) 邀请他们入队即可。

### 3️⃣ 链接格式

提交时需要填写作品访问链接，格式为：
```
https://你的域名或托管地址/...
```

你可以选择：
- **GitHub Pages**（免费，推荐）：我帮你部署
- **学校服务器**：询问指导老师
- **本地演示**：直接打开 index.html 录屏提交
